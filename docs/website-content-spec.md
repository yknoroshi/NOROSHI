## 1. プロダクトの思想と背景

### 1.1 NOROSHIとは

NOROSHI（狼煙）は、消防団の招集を確実に届ける参集・活動支援アプリ。
名前の由来は「狼煙」— 歴史的に緊急時の情報伝達に使われたシグナルファイア。

### 1.2 根本思想

中央集権的な制度が構造的限界を迎える時代に、個人と組織が自律的に判断・行動できる共通基盤を作る。

- **分散は中央集権より効率的。** 最も重要な知識は現場にある。「この道は見た目より狭い」「この消火栓は冬に凍る」。分散した暗黙知は原理的に中央に集約できない
- **分散は中央集権より強靭。** 中央集権は単一障害点を持つ。冗長で分散したシステムはストレスによってむしろ強くなる
- **分散は中央集権より人間的。** 人間は自ら判断し行動する存在。テクノロジーは奪われた判断力を取り戻す道具

### 1.3 解決する3つのボトルネック

火災出動における3つの致命的問題を解決する:

1. **気づかない** → 自動架電で確実に起こす
2. **誰が来るか分からない** → 応答ボタン＋リアルタイムダッシュボード
3. **現場で水利が分からない** → 消防特化GIS＋水利マップ

### 1.4 競合との違い

既存アプリ（FireChief、S.A.F.E.、コミュたす、消防団ワークス）はすべて「事務効率化」。
NOROSHIは戦術的な物理的解決能力に寄与する。

既存サービスは「馬車を速く走らせる工夫」。NOROSHIは「自動車の発明」。

### 1.5 開発者

広島市消防局の現役消防士がソロで開発。消火活動の物理的現実を熟知した上で設計している。
典型的なIT企業が「消防団ってこういうものだろう」と作るプロダクトとは根本的に異なる。

### 1.6 長期ビジョン（参考情報、Webサイトに直接は出さない）

- Phase 1: ソフトウェアによる浸透（現在）
- Phase 2: ハードウェア統合（小型EV、ドローン、電動ポンプ）
- Phase 3: ロボティクスの融合（自動運転EV＋自律ドローン）

最終的には「社会の自律神経系」を構築する。

## 2. コアメッセージ（トーン&ボイス）

### 2.1 キャッチコピー

| 用途 | テキスト |

| サブコピー | 招集から現場活動まで。消防団の活動を支えるアプリ。 |

| meta description | NOROSHIは消防団の招集を確実に届ける参集・活動支援アプリです。自動架電・参集状況共有・水利マップ。 |

### 2.2 トーンの原則

- **現場の言葉を使う:** 「ユーザー」ではなく「団員」。「通知」ではなく「招集」
- **プレミアムな表現:** 安売り・お得感の訴求はしない。価値で売る
- **簡潔で力強く:** 修飾語を削ぎ落とす。「すごく便利な」→「確実に届ける」
- **過度な煽りは避ける:** 「命を守る」「命に関わる」は使わない。プロの現場感で十分伝わる

### 2.3 絶対に使ってはいけない表現

| ❌ 使わない | ✅ 代わりに使う | 理由 |
|------------|----------------|------|
| マナーモード貫通 | 自動架電で確実に届ける | Critical Alertsを使わないため技術的に不正確 |
| 端末の音量設定を無視 | 電話の着信音で確実に起こす | 同上 |
| 強制通知 | 自動架電 | Critical Alertsは実装しない |
| 緊急通知（マナーモード貫通） | 自動架電による通知 | 同上 |
| 安い / お得 / コスパ | （使わない） | 安売りしない |
| 無料で十分 | 基本機能は無料 | 有料プランへの動線を殺さない |
| ユーザー | 団員 / あなた | 現場の言葉を使う |
